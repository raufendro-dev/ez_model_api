# Ez Model API

Ez (Easy) Model API adalah API mudah untuk membuat model cepat. API ini support berbagai bahasa
- Sample API
Link : http://raufendro-dev.com:3000/model (POST)

Body :

{
  "language" : "dart",
  "typeName": "user",
  "jsonString" : {"name":"david"}
  
}



## Dependencies

- Express
- Quicktype

## How to Run

- node app.js
- Link http://localhost:3000 (POST)
- Body

{
  "language" : "dart",
  "typeName": "user",
  "jsonString" : {"name":"david"}
  
}
